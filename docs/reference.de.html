Morse Referenz

Morse Zeitregeln:

  • dit = 1 Einheit
  • dah = 3 Einheiten
  • Pause innerhalb eines Zeichens = 1 Einheit
  • Pause zwischen Zeichen = 3 Einheiten
  • Pause zwischen Woertern = 7 Einheiten
